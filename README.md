# Commerces
Android application to see all commerces, can be sorted by location and filtered by category.

## Design patterns:
* Clean architecture
* MVVM

## Android jetpack:
* ViewModel + LiveData
* Data Binding
* Material Design Components(Chips, Card View, ..)
